# 🚌 Bus Reservation System

A simple console-based Bus Reservation System developed using **C Programming**. This project allows users to search destinations, reserve seats, view reservations, and cancel reservations.

---

## 📌 Features

- 🔍 Search available destinations
- 🎫 Reserve a bus seat
- 👀 View all reservations
- ❌ Cancel a reservation
- 💰 Automatic fare calculation
- 📱 Phone number validation
- 💺 Display available seats

---

## 🛠 Technologies Used

- C Programming
- Structures
- Arrays
- Functions
- String Handling (`string.h`)

## ▶️ How to Run

### Using GCC

```bash
gcc bus_reservation.c -o bus
./bus
```

### Windows (MinGW)

```bash
gcc bus_reservation.c -o bus.exe
bus.exe
```
